Detailed Description

Matrix structure that simulates a 2d matrix accessed by A(row, col).

The Matrix structure contains valuable information about the number of rows and number of columns stored in order to perform necessary logistic checks at execution time. For example, two matrices can be added to each other if and only if they are the same size; *
